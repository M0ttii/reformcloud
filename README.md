> <b>IMPORTANT NOTE:</b> ReformCloud is still in development. No stable version is available. The Full System is in beta support

### Prerequisites

To run ReformCloud you need 
 * at least 2GB Memory
 * at least 2vCores
 * Java 8

## Built With

* [Maven](https://maven.apache.org/) - Dependency Management
